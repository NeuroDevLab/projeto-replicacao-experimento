Projeto de replicação de replicação de experimento baseado no artigo <https://doi.org/10.1016/j.jbior.2022.100945> em que usaremos um desfecho e uma variável para replicar o estudo com modificações de metdologia que fazer-se-ão necessárias na implementação da pesquisa.

Para o experimento foi seleciado o desfecho comparativo de KO e controle de mutação do gene enthd2.

Levantamento inicial disponível em: <https://neurodevlab.github.io/projeto-replicacao-experimento/Inicio.html>
